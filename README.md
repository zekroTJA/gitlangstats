# gitlangstats

Usage:
```
usage: gitlangstats [-h] [--user [USER [USER ...]]] [--organization [ORGANIZATION [ORGANIZATION ...]]] [--token TOKEN] MODE

positional arguments:
  MODE                  The mode: Either size or langs.

optional arguments:
  -h, --help            show this help message and exit
  --user [USER [USER ...]], -u [USER [USER ...]]
                        The users or to analyze.
  --organization [ORGANIZATION [ORGANIZATION ...]], -o [ORGANIZATION [ORGANIZATION ...]]
                        The organizations or to analyze.
  --token TOKEN         The GitHub access token to be used.
```
